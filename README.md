# EduBooks.ai - School Finance Management

A modern school finance management dashboard built with React, TypeScript, Vite, shadcn-ui, and Tailwind CSS.

## Technologies Used

- **Vite**: Next-generation frontend tooling.
- **TypeScript**: Static typing for JavaScript.
- **React**: A JavaScript library for building user interfaces.
- **shadcn-ui**: Reusable components built using Radix UI and Tailwind CSS.
- **Tailwind CSS**: A utility-first CSS framework.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd edubooks-ui
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

### Development

Start the development server:
```sh
npm run dev
```

The application will be available at `http://localhost:8080`.

## Features

- Dashboard overview of school finances.
- Fee collection management.
- Expense tracking.
- Financial reports.
- AI-powered automation features.
